Комментирование спецификаций, плагин bubble.js

Требования

  • JavaScript
  • PHP
  • Разрешение создавать и писать файлы по всему дереву Source (TODO: в одну конкретную папку)

Установка

По умолчанию плагин находится в ядре проекта, в папке /plugins, остаётся его только включить в /user/js/options.js:

pluginsEnabled : { // other plugins bubble: true // ... }

Использование

Меню внутренней навигации пополниться слайдером, включающим режим комментирования спеки.


В режиме комментирования кликом добавляется новый баблик. Баблики сохраняются в файл и становятся доступны остальным пользователям. При закрытии баблика информация о нем стирается.
